Atividade Prática – Formulário de Cadastro

Objetivo: Praticar o uso das tags de formulário e a estrutura básica de uma página HTML.

Instruções

  1. Crie um novo arquivo chamado cadastro.html.
  2. Estruture uma página completa com as seções principais do HTML:
<html>
  <head>
    <title></title>
  </head>
  <body></body>
</html>

O tema será uma página de cadastro de clientes para uma loja fictícia.

Cabeçalho

  • Título principal: <h1> – por exemplo: Cadastro de Clientes da Loja Fábio Tech
  • Parágrafo explicando o objetivo do formulário.

Formulário

Crie um formulário usando a tag <form> com os seguintes campos:

Campo Tipo de elemento Observações
Nome completo<input type="text">Campo de texto simples
E-mail<input type="email">Valida formato de e-mail
Senha<input type="password">Campo de senha
Data de nascimento<input type="date">Seleciona data
Gênero<input type="radio">Masculino / Feminino / Outro
Interesses<input type="checkbox">Tecnologia / Jogos / Música / Esportes
Cidade/Estado<input type="text">Campo aberto
Comentário adicional<textarea>Espaço para mensagem
Enviar formulário<input type="submit">Botão de envio

Outros elementos obrigatórios

  • Use <fieldset> e <legend> para agrupar partes do formulário.
  • Use <label> corretamente associada a cada <input> (usando o atributo for).
  • Inclua uma linha divisória com <hr>.
  • Adicione uma tabela (<table>) com três clientes fictícios já cadastrados:
Nome E-mail Cidade
Maria Silvamaria@email.comPalmas - TO
João Limajoao@email.comGoiânia - GO
Ana Costaana@email.comBelém - PA

Desafio bônus (opcional)

  • Adicione um link para voltar à página inicial:
    • <a href="index.html">Voltar</a>
  • Use <abbr> ou <address> em algum ponto do código (por exemplo, o endereço da loja).

Entrega

  • Nome do arquivo: cadastro.html
  • Não será permitido o uso de inteligência artificial para realizar a tarefa!
